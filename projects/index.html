ACADEMIC PROJECTS

Insta485 (Instagram Clone), Jan - Mar 2021


my 2 teammates & I created a full-stack instagram clone from scratch.

this project had several moving parts:

1. built in REACT, the client-side application...
  • includes features such as infinite scroll and double tap to like
  • communicates with server's REST API using Javascript's fetch function

2. built using Flask and SQLite, the dynamic server-side application...
  • saves user data to our SQLite database using sha512 encryption
  • allows users to create profiles, sign-in, and sign-out using Flask sessions
  • creates separate profile, homepage, following, followers, and explore pages for each user using Flask templating and rendering
  • queries database to create JSONs for the REST API interface, also in Flask
  • supports requests for creating and deleting posts, accounts, likes, & comments
  • edits database after post and delete requests

tools used: python (Flask), SQLite, Javascript (REACT), HTML, CSS3
concepts: full-stack web applications, user sessions, jinja2-style HTML templating, REST APIs, databases, web encryption
